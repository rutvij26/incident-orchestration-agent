# Contributing

Thanks for helping improve the Incident Orchestration Agent.

## Ways to contribute
- Report bugs and edge cases
- Improve detection logic and workflows
- Add integrations or dashboards
- Improve docs and examples

## Setup
1. Install Node.js 20+ and Docker.
2. Install dependencies:
   ```
   npm install
   ```

## Run locally
```
npm run dev:stack
npm run dev:demo
npm run dev:agent
npm run run
```

## Tests
```
npm run healthcheck
npm run test
```

## Pull requests
- Keep changes focused and small.
- Include a short summary and test plan.
- Update docs when behavior changes.
